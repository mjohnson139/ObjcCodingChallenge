# ObjcCodingChallenge

Objective-C coding challenges.

## Requirements

To run the Objective-C coding challenges, you'll need:

- Xcode 13 or later

## Getting Started

1. Clone the repository
2. Open ObjcCodingChallenge folder in Xcode
3. Build and run the tests

## Challenges

The challenges are located in the `Sources/ObjcCodingChallenge` directory. Tests are located in the `Tests/ObjcCodingChallengeTests` directory.

Implement solutions to the challenges and ensure the tests pass.
